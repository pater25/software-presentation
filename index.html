01 — Definition

What Is
Software?

Physical
Hardware
= physical components
Logical
Software
= instructions & programs

Software is the set of instructions, data, and programs that tell a computer how to work. Unlike hardware, software is not physical: it is the logical part of a computer system, invisible yet essential for everything the machine does.

02 — Categories

Types of
Software

System Software
Management & Control

System software manages the computer's resources and provides the foundation on which application software runs. It operates in the background, coordinating hardware components and allocating resources efficiently.

⚙️ Operating Systems 🔧 Device Drivers 🛡️ Security Software 🖥️ Firmware
Application Software
Tasks & Everyday Use

Application software helps users perform specific tasks. It runs on top of the operating system and uses system resources to deliver functionality directly to the end user, from browsing the web to editing videos.

In computer architecture, hardware is the base layer — the physical foundation everything else is built upon.

The operating system sits directly above hardware, managing its resources and providing services to applications.

The user interacts mainly with applications and the user interface provided by the system, rarely engaging directly with lower layers.

04 — Operating System

Operating System:
Definition

Hardware
↓
Operating System
↓
Applications

An operating system is system software that manages hardware resources and provides services for programs.

It is the bridge between the physical machine and the software used by the user — without it, applications would have no standardised way to access hardware.

09 — Licences

Software
Licences

A software licence is a legal agreement that defines how software can be used, copied, distributed, or modified. Licences define user rights, restrictions, and responsibilities — they protect both developers and users.
